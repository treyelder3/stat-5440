# STAT 5440: Applied Bayesian Statistics

Course repository for STAT 5440. Contains lecture code, datasets, and final project materials.

## Repository Structure

```
├── lectures/       # Code and materials organized by lecture
├── datasets/       # Small datasets used throughout the course
└── final-projects/ # Group final project submissions by semester
```

## Lectures

The `lectures/` folder contains code files organized by lecture topic. Each lecture may include R scripts, R Markdown files, or Jupyter notebooks demonstrating Bayesian methods covered in class.

## Datasets

The `datasets/` folder contains small datasets used in lectures and assignments. Larger datasets should be hosted externally and documented with a link in the relevant lecture folder.

## Final Projects

Group final projects are organized by semester under `final-projects/`. See the submission guidelines in each semester's folder for upload instructions.
